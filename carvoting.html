About This Project

This Car Voting Application was developed as a portfolio project to showcase my front-end development skills. The application features:

  • Interactive carousel with auto-play/pause functionality
  • Voting system with persistent data using localStorage
  • Dynamic leaderboard with percentage calculations
  • Interactive charts using Chart.js
  • Responsive design that works on all devices
  • Clean, semantic HTML and modular CSS
  • Well-structured JavaScript with clear functions

The project demonstrates my ability to create engaging, interactive web applications without relying on frameworks, highlighting my understanding of core web technologies.
